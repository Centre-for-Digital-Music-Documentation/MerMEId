Create a music anthology

A distinction should be made between anthologies regarded as one work and anthologies regarded as a collection of individual works.

Anthologies regarded as one work, e.g. certain song cycles, are simply works with a number of constituent components. These components are defined at the Music tab:

  • Near the bottom of the page, add the desired number of components.
  • Save your work and start editing the components one at a time by clicking the component's edit button.
  • Enter the details about the component, such as the title and the text author.
  • Click the "save and close" button to save your work and return to the Music tab.

Anthologies of individual works can be created as follows:

  • First, create and save the individual works. However, sources representing the entire anthology should not be defined at this stage.
  • Then, create another work to represent the anthology. At the Work tab, start defining relations to the contained works by clicking the "Add relation" button.
  • Select the "contains:" relation
  • If you know the work's file name, you may enter it directly in the [target?] field.
    Otherwise, open the works browser by clicking the [link?] button. In the works browser, locate and click the desired work. It may be helpful to narrow the search to a specific file collection or enter search terms such as the title. Please note that the file browser may need some time to update.
  • Repeat this with each of the contained works and save your work.
  • Add any other desired information about the anthology, such as title, source descriptions etc.

Remember that relations are bidirectional. In order to create links both ways (from the anthology to the individual works and back), edit each of the contained works to add a relation back to the anthology. You have easy access to the contained works by using the "Go to..." button
