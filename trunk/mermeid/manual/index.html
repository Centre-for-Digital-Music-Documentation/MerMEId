MerMEId basics

MerMEId (Metadata Editor and Repository for MEI data) is a tool for editing, (pre-)viewing and handling of music metadata based on MEI.

It consists of three main components: the MEI file browser, the metadata editing tool, and the HTML preview.

File list

The usual way to start working with MerMEId would be to open the file list, usually located at http://[your hostname here]/storage/list_files.xq (here). You should then see a list of existing files in your collection (if any):

Collection buttons (in this example, "HartW", "CNW", "SchM", and "GW") will be visible only if you have spefcified such collections in your files. Clicking them will reduce the list to show only files belonging to that collection.

You may also reduce the number of files displayed using the "Search" field in the upper right corner. Navigate through the result pages using the page numbers or the "Previous page" / "Next page" links.

From the list, you have the following options:

  • Add new file: Clicking the "Add new file" button will open a new file in the metadata editor
  • HTML preview: Click the title of any file to see its metadata displayed as formatted HTML.
  • XML source: Click the XML view icon to see file's the plain XML data
  • Edit file: Click the edit icon to open the file in the metadata editor
  • Delete file: Clicking the delete icon will display a warning that you are about to delete the selected file. Click "OK" to delete, otherwise "Cancel".

HTML Preview

The purpose of the HTML preview is to give you an overview of the data you have entered. Your project may or may not use it as a basis for public display of your data as well, but probably this is not how your data will be displayed in the end.

If titles or other text is given in more than one language, all other alternatives than the first one are shown in grey with an abbreviated indication of the language at the beginning. To hide these alternative language texts, click the "Hide alternative languages" link in the upper right corner.

By default, only the headings of the main sections are shown. To view the contents, click the the expand icon or the heading. To hide the contents, click the heading again. The number of headings shown depends on which sections have any content.

At the bottom of the page, in grey, metadata about the file are displayed (i.e. the colophon).

Hints to using the editor

Entering special characters

Unicode characters may be pasted into your text. Please note, however, that displaying special characters correctly may depend on the fonts installed on the user's computer and/or on the way your data are processed for display. Especially musical symbols are still very poorly supported by font suppliers (at present, only flat, natural, and sharp signs can be used fairly safely).

A list of selected unicode characters is given at the end of this document; symbols may be simply copied from there.

  • Important: Never use the ampersand sign (&) like you would in a text document, as it has a special meaning in XML. If you need an ampersand in your text, enter it as &amp; – this is the standard code for the ampersand sign. Otherwise you will get an XML error when trying to view your file. For example, instead of writing Breitkopf & Haertel, write Breitkopf &amp; Haertel.

Entering dates

Since dates in MEI consist of a visible and a hidden (machine-readable) component, a special date editor is opened whenever you click a date field or the "Edit" button next to a date.

The visible part of the date is entered at the bottom. You may enter anything you would like to be displayed as the date, e.g. "May 23", "1906-1918", "2012-09-19" or "No later than the composer's 41st anniversary".

The machine-readable part is entered using the day, month and year fields in the middle. You may either enter a single date or a date interval. Choose the appropriate setting at the top. The machine-readable part always uses exact dates (i.e. both day, month and year), but for your convenience the date editor computes exact dates from incomplete dates entered. For instance, if you choose "Date interval" and enter only the two years 1895 and 1896, these will be interpreted as an interval beginning 1 January 1895 and ending 31 December 1896 when clicking the "OK" button. If you choose the months as well, the first day of the month will be added in the start date and the last day of the month in the end date automatically (not taking into account leap years, though!). You can even enter an interval of a year or a month by selecting "Single date" and filling in only the year or the year and a month. An interval will be computed when you click the "OK" button.

Once computed, the machine-readable part will be displayed on the form in square brackets for your control.
