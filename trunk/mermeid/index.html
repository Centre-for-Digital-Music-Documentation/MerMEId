Metadata Editor and Repository for MEI Data

MerMEId is a system for the editing, handling, and (pre-)viewing of music metadata, based on the the MEI (Music Encoding Initiative) XML schema. MerMEId is developed at the Danish Centre for Music Publication (DCM) both for the production of thematic catalogues of works and for organizing source- and work-related information during the preparation of scholarly editions of music.

This is a preliminary project site for the MerMEId software with a demo installation. 

Read more about the project at DCM's web site.

Try it!

The demo installation has been tested with Internet Explorer 8, Firefox 18.0, and Google Chrome 24.0 on Windows systems, and with Firefox on Linux, but compatibility is mainly dependent on the third-party software used (see the installation guide below for details). The user interface is optimized for Firefox.

In order to do things you'll need the user ID and password. They are honoured and guest, respectively. Welcome :-)

To get started, you may want to have a look at the manual (opens in a new window or tab). 

What you can do with the demo

You can create new files and change everything you like in the existing ones. Do not save anything indispensable, or even expect that a document created one day will survive the night. The system is restored daily. However, you can download your work to your own file system by right-clicking the XML code view icon and selecting "Save as...".

The demo includes a number of sample files illustrating various aspects of working with music metadata.
