– Metadata Editor and Repository for MEI Data

In connection with the preparation of a thematic catalogue of Carl Nielsen's Works (CNW), DCM is developing a system for the editing and handling of music metadata, based on the the MEI (Music Encoding Initiative) XML schema. The system, which has been named Metadata Editor and Repository for MEI Data (MerMEId).

This is a preliminary project site for the MerMEId software, with a demo installation.

  • Try MerMEId. In order to do things you'll need the User ID and password. They are honoured and guest, respectively. Welcome.
